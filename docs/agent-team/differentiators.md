# Differentiators

Delivery Kit is built around delivery efficiency, not broad autonomy.

## Distinctive choices

- **Lane-based workflow:** fast, standard, protected.
- **Artifact-first handoffs:** `.agent/` files carry context between steps.
- **Acceptance-driven QA:** every QA result maps to a criterion.
- **Risk-triggered depth:** security/performance/release gates run when warranted.
- **Compact root prompt:** shared rules stay small.
- **Installable source of truth:** maintain one kit and sync it into app repos.
- **Human approval at boundaries:** product scope, protected risk, release.

## Philosophy

Evidence beats confidence. Small stories beat giant tasks. Routing beats one-size-fits-all process.
